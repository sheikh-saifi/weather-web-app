# Weather Web App
A simple, responsive weather web application that displays current weather information for any city worldwide.

## 🌤️ Live Demo
Check out the live app: [Weather Web App](https://sheikh-saifi.github.io/weather-web-app/)

## 📋 Features

- Search weather by city name
- Displays temperature, humidity, wind speed, and weather conditions
- Shows weather icons based on conditions
- Responsive design for mobile and desktop
- Real-time weather data from OpenWeatherMap API

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## 🚀 How to Use

1. Enter a city name in the search box
2. Click the search button or press Enter
3. View the current weather information for that city
